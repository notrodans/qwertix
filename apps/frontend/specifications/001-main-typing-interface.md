# Main Typing Interface

## 1. Goal
Create a minimalistic typing interface inspired by modern typing apps, focused on clarity and focus.

## 2. Visual Requirements
- **Theme:** Dark mode by default.
- **Typography:** Monospaced font for perfect alignment.
- **Layout:**
    - Centered focus area.
    - Configuration summary at the top (e.g., Mode).
    - Dynamic text display in the middle.
    - Control bar at the bottom (e.g., Restart).
- **Colors:**
    - Distinct states for characters: Untyped, Correct, Incorrect.

## 3. Acceptance Criteria
- [ ] Interface is perfectly centered.
- [ ] Text block is legible and uses monospaced font.
- [ ] Visual indicators for current progress and active word are visible.
- [ ] Controls are accessible via mouse and keyboard.
