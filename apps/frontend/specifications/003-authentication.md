# Authentication & Identity

## 1. Overview
Manages user access, identity display, and historical data persistence.

## 2. User Interface
- **Login**: Credential entry for registered users.
- **Admin Actions**: Interface for authorized administrators to create new accounts.
- **Header Identity**: Displays current username and provides logout options.

## 3. Experience for Authenticated Users
- Automated saving of race results and performance metrics.
- Access to personal typing history and progression over time.
