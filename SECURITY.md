# Security Policy

## Supported Versions

Currently, we provide security updates only for the latest stable release.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly. **Do not create a public issue.**

Please report any security-related issues by emailing **notrodans@gmail.com**.

We will acknowledge your report within 48 hours and provide a timeline for a fix if the vulnerability is confirmed. We ask that you do not disclose the vulnerability publicly until we have had a chance to address it.

Thank you for helping keep this project secure!
